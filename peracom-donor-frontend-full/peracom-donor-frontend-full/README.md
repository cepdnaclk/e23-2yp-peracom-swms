# PeraCom Donor Frontend

## Run

```bash
npm install
copy .env.example .env
npm run dev
```

## Backend expectation

This frontend expects your backend  at:

```env
VITE_API_BASE_URL=http://localhost:5000/api
```

## Pages included
- Login / Register
- Dashboard
- Scholarships
- Request Scholarship
- Approved Students
- Student Profile
- Progress Updates
- Report Issue
